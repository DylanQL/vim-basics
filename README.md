## Introduccion a vim

Vim por defecto se encuentra instalado en casi todos los sistema basados en UNIX, si eres usurio Windows debes de descargarlo.
 
## Como entrar y salir de Vim
 
Para entrar en Vim solo se debe de poner > vim en la terminal
 
 ```
vim
```
 
Pasa salir de Vim se debe de colocar lo siguiente
```
:q
```
Si en caso el anterior comando te da algùn tipo de error,seguramente es porque editarse sin querer el archivo, en ese casos puedes podrar el siguiente comando
```
:q!
```
